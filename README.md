# Nextjs OpenJira App 
needs db to run locally 

``` docker-compose up -d ```

* -d = detached

* Local URL

``` mongodb://localhost:27017/entriesdb ```

## Environment variables from the ``` .env.template ``` file needs to be configure

## Call the database with development info with

``` http://localhost:3000/api/seed ```
